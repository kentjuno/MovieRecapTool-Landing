Bước 1 - Cài đặt OmniVoice API Server

Môi trường thuyết minh giọng nói cục bộ (chạy offline trực tiếp trên máy hoặc mạng LAN của bạn).

📥 Tải trực tiếp file ZIP (Khuyên dùng) Xem mã nguồn trên GitHub
  • Cách tải dễ nhất: Click vào nút "Tải trực tiếp file ZIP" màu xanh ở trên. File zip chứa toàn bộ server sẽ được tự động tải xuống máy tính của bạn ngay lập tức.
  • Nếu bạn click vào liên kết xem trên GitHub, hãy chọn nút Code màu xanh lá cây ở góc phải rồi click Download ZIP như hình minh họa bên dưới để tải về máy:
  • Sau khi tải xong, nhấp chuột phải vào file omnivoice-api-server-main.zip và chọn Extract All (Giải nén tất cả) ra một thư mục.
  • Lần đầu chạy: Nhấp đúp (Double-click) vào file setup_and_run.bat trong thư mục vừa giải nén để tự động tải các gói mô hình giọng nói AI và cấu hình môi trường. (Quá trình này chạy tự động hoàn toàn, bạn chỉ cần đợi cửa sổ chạy xong).
  • Các lần chạy sau: Chỉ cần nhấp đúp file run.bat để bật API Server lên trước khi mở tool recap phim.

Bước 2 - Cài đặt Antigravity CLI

Công cụ dòng lệnh CLI để xác thực và kích hoạt môi trường trí tuệ nhân tạo Google Pro.

Mở PowerShell (với quyền Run as Administrator nếu có thể) và dán dòng lệnh sau:

irm https://antigravity.google/cli/install.ps1 | iex

Mở CMD và dán dòng lệnh sau:

curl -fsSL https://antigravity.google/cli/install.cmd -o install.cmd && install.cmd && del install.cmd

Sau khi tiến trình cài đặt hoàn thành, hãy mở một cửa sổ CMD mới và chạy lệnh sau:

agy

Hướng dẫn đăng nhập tài khoản Google Pro/Ultra chi tiết từng bước:

  1. Sau khi gõ agy và bấm Enter, màn hình CMD sẽ hiển thị một đường link xác thực kèm theo mã thiết bị.
  2. Bôi đen đường link đó (NHỚ BÔI ĐEN TOÀN BỘ ĐỦ ĐƯỜNG LINK BAO GỒM HTTPS:// Ở ĐẦU TỚI CUỐI!!!), nhấp chuột phải để Copy (trên Windows CMD, chỉ cần bôi đen rồi click chuột phải là máy sẽ tự copy). Dán link này vào trình duyệt web của bạn.
  3. Tiến hành đăng nhập tài khoản Google của bạn trên trình duyệt (Khuyến nghị sử dụng tài khoản Google Pro hoặc Ultra trở lên để AI có hiệu năng phân tích kịch bản tốt nhất).
  4. Sau khi đăng nhập và cấp quyền thành công, trình duyệt sẽ trả về một mã code xác thực.
  5. Copy mã code đó, click chuột phải vào cửa sổ CMD để dán (paste) và nhấn Enter để hoàn tất liên kết tài khoản.
  6. Gõ thử 1 hoặc 2 câu chat bất kỳ để kiểm tra xem AI có phản hồi bình thường hay không.

Bước 4 - Cập nhật Driver NVIDIA & Cài CUDA Toolkit

Yêu cầu tối quan trọng để tăng tốc phần cứng (GPU) cho Whisper (nhận diện giọng nói nhanh gấp 10-20 lần CPU) và xử lý video.

🌐 Tải NVIDIA Driver mới nhất 🛠️ Tải CUDA Toolkit
  • NVIDIA Driver: Đảm bảo máy tính của bạn đã cập nhật Driver card đồ họa NVIDIA mới nhất từ link chính thức bên trên.
  • CUDA Toolkit: Tải và cài đặt CUDA Toolkit phiên bản mới nhất. Khi cài đặt, hãy chọn cài đặt mặc định (Express Install).
  • Kích hoạt thư viện CUDA/cuDNN cho Whisper:
    • Khởi chạy MovieRecapTool, ứng dụng sẽ quét hệ thống và hiển thị Banner màu vàng ở màn hình đầu tiên: "Phát hiện card NVIDIA nhưng chưa kích hoạt CUDA/cuDNN".
    • Nhấp vào nút "Kích hoạt GPU (Tải tự động)" bên cạnh để ứng dụng tự động tải về cuBLAS & cuDNN (dung lượng ~270MB), giải nén và kích hoạt GPU hoàn hảo.
    • Nếu trong app gặp lỗi mạng không tải được, hãy vào thư mục cài đặt ứng dụng và click đúp vào file activate_gpu.bat để chạy kích hoạt thủ công.
    • Sau khi kích hoạt xong, hãy tắt và khởi động lại ứng dụng để Whisper nhận dạng và hoạt động trên GPU NVIDIA.

FAQ trước khi nhắn hỗ trợ.

Các lỗi phổ biến nhất khi app mới cài: SmartScreen, voice server chưa chạy, license khác máy.

Windows báo SmartScreen thì sao?

Bản hiện tại chưa code-sign bằng certificate thương mại, nên Windows có thể hiện cảnh báo lần đầu. Chọn More info rồi Run anyway nếu bạn tải từ link chính thức.

Không có phụ đề thì dùng được không?

Dùng được. MovieRecapTool có thể dùng Whisper để trích xuất lời thoại từ audio của phim. Nếu có sẵn file SRT chuẩn thì nên nhập vào để tiết kiệm thời gian phân tích.

Có cần chạy OmniVoice trước không?

Có. Trước khi render giọng đọc, hãy mở omnivoice-api-server và chạy run.bat. Nếu OmniVoice chạy ở máy khác, vào CẤU HÌNH API & OMNIVOICE trong app để đổi URL server.

Máy không có NVIDIA GPU có dùng được không?

Có, nhưng sẽ rất chậm, đặc biệt ở bước Whisper lấy phụ đề và các bước xử lý AI/audio. Nếu làm recap thường xuyên, nên dùng máy có NVIDIA GPU và đã kích hoạt GPU trong app.

Whisper báo thiếu CUDA dù máy đã có CUDA?

CUDA/driver trên Windows chưa đủ. MovieRecapTool còn cần DLL cuBLAS/cuDNN cho faster-whisper. Hãy bấm Kích hoạt GPU (Tải tự động) trong app hoặc chạy activate_gpu.bat trong thư mục cài đặt, sau đó mở lại app.

Render ra tiếng "te te" hoặc không gọi OmniVoice?

Thường là OmniVoice server chưa chạy hoặc URL sai. Mở CẤU HÌNH API & OMNIVOICE, kiểm tra URL, rồi chạy lại server.

License nhập ở máy khác được không?

Mỗi license được bind với 1 máy sau lần kích hoạt đầu tiên. Nếu cần đổi máy, liên hệ admin để xử lý.

Trial 30 ngày nâng lên lifetime có được trừ tiền không?

Có. Nếu nâng lên lifetime trong thời gian trial 30 ngày, 400K trial được trừ 100%, bạn chỉ cần trả thêm 499K. Khi nâng cấp, nhắn Facebook Kent Juno kèm license hoặc thông tin mua trial để được cấp/nâng license chính thức.

Pass 7 ngày có được trừ vào lifetime không?

Không. Gói 100K / 7 ngày chỉ dùng để test nhanh máy và workflow, mặc định không được trừ vào lifetime nếu mua sau.
